Tags HTML: Estruturas HTML

<>               abertura 
</>              fechamento

<!doctype>       tipo de documento HTML utilizado
<html> </html>   tag html da estrutra de toda a página web.
<head>           Parte que não é exibida no navegador da Web quando a página é carregada ( Animações , interações com o usuário)
<title>          Título da Página
<meta charset="utf-8">  Define que você pode colocar assentos na página web
<body>           Informações que aparecem diretamente para o usuário ( background= fundo da página)
Tags de Cabeçalho:
<br> Quebra de texto
<h(número)> tag de cabeçalho

Tags de Parágrafos:
<p> Paragráfo , agrupar linhas e definir
